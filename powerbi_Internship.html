Weekly Internship Roadmap

Week 1: Project & Data Identification

Objective: Define the project scope, select an industry, identify business questions, and acquire initial raw data.

Key Tasks:

  • Brainstorm and select an industry for analysis.
  • Define 2-3 key business questions to answer.
  • Explore public data sources and acquire 2-4 raw datasets.
  • Perform initial data profiling to understand structure and quality.

Recommended Data Sources:

Kaggle Data.gov UCI Repo World Bank

Week 2: Data Storage & Infrastructure

Objective: Select and set up a suitable data storage solution using free trial options, and ingest the raw data.

Key Tasks:

  • Evaluate and select a cloud or local database solution.
  • Set up accounts and required services (e.g., database, storage).
  • Load raw datasets into the chosen data source.

Free Trial Platforms:

Azure SQL DB AWS RDS GCP BigQuery SQL Server Dev

Week 3: Data Transformation & Modeling

Objective: Clean, transform, and structure the raw data into a star schema suitable for analytical reporting.

Key Tasks:

  • Clean data: handle missing values, duplicates, and errors.
  • Transform data: create calculated columns, filter, and aggregate.
  • Design a star schema with fact and dimension tables.
  • Implement the data model in your chosen data source using SQL or other tools.

Core Skills:

Data Cleaning ETL Processes Dimensional Modeling SQL Scripting

Weeks 4-5: Power BI Report Building

Objective: Connect Power BI to the prepared data source, create the data model, and build an interactive, insightful report.

Key Tasks:

  • Connect Power BI to the data source and load tables.
  • Define relationships in the Power BI data model.
  • Write DAX measures to calculate KPIs.
  • Design and build multiple interactive report pages.
  • Implement slicers, filters, and drill-throughs for exploration.
  • Focus on clear storytelling through visuals.

Focus Areas:

DAX Data Visualization UI/UX Design Dashboard Interactivity

Week 6: Finalize & Document

Objective: Refine the report, prepare project documentation, present findings, and explore advanced concepts.

Key Tasks:

  • Review and refine the report based on feedback.
  • Create comprehensive project documentation.
  • Prepare and (optionally) deliver a final presentation.
  • Explore advanced topics like RLS and Power BI Service.

Advanced Topics:

Row-Level Security Power BI Service Data Refresh Custom Visuals
